Projects

This section of the website, as you may have guessed, contains all the information about projects I'm currently working on as well as projects I have completed in the past. There are links to view the source code as well as to download the projects.

Minecraftify

Minecraftify is an Android application that uses an image processing algorithm to break up an image taken by the user into Minecraft blocks. Brandon Caudell , Adam Heald , and myself created the application in 36 hours at Hack ISU in Fall 2014. Minecraftify is currently published on Google Play and has over 1500 downloads. Source code is available on GitHub. Written in Java and XML.

Select-A-Major

Select-A-Major is a web application that helps people search for colleges that offer the majors they're interested in. The development version of the application is live here. Currently only colleges in Missouri can be searched. Source code is available on GitHub. Written in HTML, JavaScript, CSS, and Bootstrap, with Firebase as a backend.

A to Zoo

This web application lets people for which zoos have their favorite animals. Select zoos across the United States are included. This app was built by me, Peter Dirks , and Robert Stovall at Hack Illinois in Spring 2014. Source code is available on GitHub. Written in HTML, JavaScript, CSS, and Bootstrap, with a MySQL backend.

LearnLisp

LearnLisp is a fully functioning lisp style (Polish prefix notation) programming language. I built this programming language with help from this guide. Source code is available on GitHub. Written in C.

Survive Calculus

The first Android app I developed is Survive Calculus, which contains basic calculus information and formulae. The app is also published on Google Play. Source code is available on GitHub. Written in Java and XML.
